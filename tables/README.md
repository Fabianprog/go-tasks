# Tabellen-Aufgaben

In diesem Verzeichnis finden sich verschiedene Aufgaben, die sich damit befassen, wie
man Werte in Tabellen (Ein- und Zweidimensionale Arrays) findet.

## Herangehensweisen

Es geht los mit einfachen Aufgaben, die auch schon aus anderen Übungen bekannt sein könnten.
Darauf aufbauend werden die Aufgaben dann komplexer.

Es geht hier darum, entweder die Lösungen der vorherigen Übungen zu wiederholen und anzupassen,
oder zu erkennen, wie man die Funktionen aus den vorherigen Übungen wiederverwenden kann,
um neue Aufgaben zu lösen.

## Aufgaben-Übersicht

Die Aufgaben können in folgender Reihenfolge bearbeitet werden:

| Aufgabe    | Beschreibung                                                    |
| ---------- | --------------------------------------------------------------- |
| find       | Einfache Suche in einer Liste                                   |
| lookup     | Suche in einer Liste, Ergebnis aus einer anderen Liste liefern  |
| get_column | Werte einer bestimmten Spalte aus einer Tabelle extrahieren     |
| vlookup    | Suche in einer Tabelle (ähnlich der SVERWEIS-Funktion in Excel) |
